    SQLite3 Editor

## Vantagens

- Não precisa configurar
- Schema simples
- Rápido e Estável
- Utilizado/Disponível em diferentes aplicações
- 90% similar ao MySQL, PostgreSQL e Outros.

## Desvantagens

- Não dá para restringir acessos
- Não foi criado para receber muitas escritas por segundo
- Banco de dados deve estar próximo ao Back End

### OBS.: A Linguagem SQLite não é caseSensitive --> porém, como boa prática os comandos são escritos em caixa alta
