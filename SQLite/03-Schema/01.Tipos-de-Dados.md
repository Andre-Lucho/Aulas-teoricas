## INTEGER e TEXT

Esses são os dois tipos de dados mais comuns no SQLite.

### INTEGER

Armazena números inteiros:

- inteiros,
- boolean (0,1),
- timestamp

### TEXT

Armazena textos

- strings,
- datas,
- json

## Dados no SQLite

Existem 3 momentos importantes em que o SQLite lida com tipos de dados:

1. Definição de Tabelas

    Definir o tipo de dado da coluna.

2. Inserção de Dados

    Converte o dado para o tipo de acordo com a afinidade da coluna.

3. Consulta de Dados

    Retorna o valor armazenado.

## Tipos de Dados

O SQLite possui os seguintes tipos de dados:

1. TEXT

    Texto

2. INTEGER

    Números inteiros

3. REAL

    Números flutuantes: double, float

4. BLOB

    Dados binários

5. NULL

    Valor nulo

## Afinidade de Tipo

SQLite **não impõe** tipos de dados estritos, mas sim uma _afinidade de tipo_.  
Isso significa que você pode armazenar qualquer tipo de dado em qualquer coluna, mas o SQLite tentará converter os dados para o tipo mais apropriado, de acordo com a afinidade da coluna.

### São 5 tipos de afinidade:

1. TEXT
2. INTEGER
3. REAL
4. BLOB
5. NUMERIC: Números inteiros ou flutuantes

https://www.sqlite.org/datatype3.html

## Conversão de Tipos

O SQLite converte automaticamente os tipos de dados quando necessário. Por exemplo, se você inserir um número em uma coluna TEXT, o SQLite armazenará esse número como texto.
