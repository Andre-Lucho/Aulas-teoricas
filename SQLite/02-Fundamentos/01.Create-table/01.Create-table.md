# Create Table
Cria uma tabela cursos e define as colunas (id, nome, aulas). <br>Sempre coloque o ';' no final da instrução SQL.

```sql
CREATE TABLE cursos (id, nome, aulas);
```

## Drop Table
Remove a tabela cursos

```sql
DROP TABLE cursos;
```

## Definir Dados
Define o tipo de dado da coluna e restrições (constraints)

```sql
CREATE TABLE cursos (
  id INTEGER NOT NULL,
  nome TEXT NOT NULL,
  aulas INTEGER
);
```

## Checar Tabela
Verifica as características da tabela cursos

```sql
PRAGMA TABLE_INFO('cursos');
```
