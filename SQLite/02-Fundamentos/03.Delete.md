## Delete
Deleta todas as linhas da tabela cursos

```sql
DELETE FROM "cursos";

```

## Where
Deleta as linhas da tabela cursos onde o id é igual a 1.

```sql
DELETE FROM "cursos" WHERE "id" = 1;

```

**IMPORTANTE:**  
- Colocar condicional "WHERE" , pois, sem ela, todos os elementos da tabela serão deletados.
