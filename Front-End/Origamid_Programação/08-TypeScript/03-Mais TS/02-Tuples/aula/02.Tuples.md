# Tuples

### Tuples são arrays que possuem dados em posições fixas.

```
const produto1: (string | number)[] = ['Notebook', 2500];
const produto2: [string, number] = ['Notebook', 2500];

const nome1 = produto1[0]; // string | number
const nome2 = produto2[0]; // string

const [nome, preco] = produto2;

```

<br>

## as const

### Torna um dado readonly e infere o tipo de dado mais específico possível. <br>Em métodos que retornam Array's, as mesmas são transformadas em Tuples.

```
function getText(selector: string) {
  const el = document.querySelector<HTMLElement>(selector);
  if (el) {
    return [el, el.innerText] as const;
  } else {
    return null;
  }
}

const button = getText('button');

if (button) {
  const [buttonElement, buttonText] = button;
}


```

## MTO Importante - uso de Tuplas no tratamento de API's:

### Tuplas ajudam na manipulação de Arrays retornados de API's:

Vantagem de API's que retornam arrays --> _Tamanho do arquivo armazenado no servidor:_<br> se o retorno é sempre um mesmo tipo de elemento, não há necessidade de retornar um objeto, pois os elementos são os mesmos e ocupam os mesmo lugares --> aqui entra a importância das Tuplas --> tipos na mesma sequência!!
