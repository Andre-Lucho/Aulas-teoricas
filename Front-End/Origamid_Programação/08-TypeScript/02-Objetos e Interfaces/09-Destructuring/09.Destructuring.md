# Destructuring

## Durante a desestruturação de objetos, podemos indicar o tipo de dado com a sintaxe: { key1, key2 }: { key1: type1; key2: type2; }

```
document.body --> quero pegar o body:

const { body }: { body: HTMLElement } = document;
```

### Com objetos:

---

```
function handleData({ nome, preco }: { nome: string; preco: number }\*) {
nome.includes('book');
preco.toFixed();
}

handleData({
nome: 'Notebook',
preco: 2000,
});

*Obs: é a interface do objeto
```

### Se tiver muitos dados no meu objeto e quero desestruturar apenas alguns, uso a Interface:

```
interface Produto{
nome: string,
preco: number,
fabricante: string,
anoFabricacao: number,
garantia: boolean,
retirada: boolean,
}

function handleData({ fabricante, retirada }: Produto) {
fabricante.includes('book');
retirada.valueOf;
}

const obj ={
nome: 'Notebook',
preco: 2000,
fabricante: 'Sansung',
anoFabricacao: 1998,
garantia: true,
retirada: true,
};

handleData(obj);
```

## Conhecer os Dados

### Durante a desestruturação é necessário indicar o tipo exato do dado esperado pelo TypeScript. Ex: um currentTarget pode ser EventTarget | null.

```
function handleClick({
currentTarget,
pageX,
}: {
currentTarget: EventTarget | null;
pageX: number;
}) {
if (currentTarget instanceof HTMLElement) {
currentTarget.innerHTML = `<h1>Mouse Click em x:${pageX}</h1>`;
}
}

document.documentElement.addEventListener('click', handleClick);
```

### \*\*IMPORTANTE: Caso de erro na função de callback com desestruturação --> ver se a herança do DOM está correta para o elem a ser desestruturado

## ...rest

### O operador ...rest retorna uma Array.

```
function comparar(tipo: 'maior' | 'menor', ...numeros: number[]) {
if (tipo === 'maior') {
return Math.max(...numeros);
}
if (tipo === 'menor') {
return Math.min(...numeros);
}
}

console.log(comparar('maior', 3, 2, 4, 30, 5, 6, 20));
console.log(comparar('menor', 3, 2, 4, 1, 5, 6, 20));
```
