2. PQ o as letras maiores e de cores diferentes no corpo do site?
Principal intenção do desing aqui é vender o serviço → chamando a atenção do usuário para o tempo que ele irá poupar utilizando o serviço e a facilidade em utilizá-lo

3. Side-nav a direita → Registro = também em destaque

c) Pq apenas 4 campos p/ preenchimento do cadastro? → + dinamismo ao preenchimento == + adesão ao site(pesquisa)

2. Ícones no estilo linha e não preenchido → mais clean (destaque) == pouco espaço
