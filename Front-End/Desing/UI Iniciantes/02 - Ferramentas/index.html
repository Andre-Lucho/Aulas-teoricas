Ferramentas de Prototipagem

Aula - Navegação

Atalhos

● Espaço + click esq → Movimentação entre os frames/ elems do frame no design
● 'Crtl' + scroll ou ' + / - ' → Zoom + / -
● 'Z' + click → Zoom detalhado
● 'Alt' → Ver os espaçamentos entre frames
● 'Crtl' ou 'Shift' + cantos do elemento → Rotação do elem
● Elemento + número → Opacidade do Elem

Ferramentas

● 'V' ou Esc → Mover
● 'K' → Escala - Escalar a imagem conforme sub-ferramentas à direta
● 'F' → Frame
● 'R' → Retângulo
'R' + 'Shift'→ Força a criação de quadrados
● 'L' → Linha
'L' + 'Shift'→ Força a criação de linhas em ângulos de 45° (45°, 90°...)
● 'O' → Círculo
'O' + 'Shift'→ Força a criação de círculos perfeitos
● 'I' → Color Peaker
● 'P' → Pen Tool → Criação de formas com linhas

● Criação de Elemenento Clone → 'Alt' + clicar e arrastar

'Enter' ou 2 cliques na forma → Visualizar os pontos de formação dela
'Crtl' + click → Modificação dos pontos em formas curvas
'Crtl' + click + 'Alt'→ Modificação em apenas 1 das alças da figura
** 'Enter' em cima de uma forma já criada → Habilita o Pen Tool para a forma Pencil → Criação de formas livres

** Posso ver todos os atalhos no botão '?'

Aula - Formas

Atalhos


● Barra lateral direita:

Alinhamento:

Alinhamento das forma vertical e horizontalmente

Última opção → Distribui o alinhamento de forma proporcional ( props FlexBox e Grid)

Dimensões:

Além dos valores, posso também inserir cálculos! Ex. 200+20 → 220px

Opacidade

Valor em percerntual no sub-menu - Layer
● Posso definir com valores numéricos no teclado

Fill(Preenchimento):

Preenchimento da forma → parte interna - cores

Stroke(Borda):

Bordas

Effects(Sombras):

Sombras → Simulação de profundidade (3D) = De onde o sol está incidindo no objeto!

Ícone sol = Configurações da sombra:

X: Add sombra horizontal → + == p/ a direta ; - == p/ a esquerda (crescimento da sombra)
Y: Add sombra vertical → + == p/ a baixo ; - == p/ a cima

*Blur → 'Embasamento' → mais próx 0 == 3D ao elemento (sólida);
números maiores → Sombra;
números ainda maiores → Desfoque.
*Spread → o tamanho da sobra aumenta além do tam do elemento

** Drop Shadow → efeito de profundidade (sobreposição)
** Inner Shadow → efeito de profundidade (p dentro)

● Barra lateral Esquerda:

Elemento + cadeado → bloqueia a forma

'Shift' + click → Seleção de vários elementos
'click dir' → Várias opções
Crtl' + G → Group Selection → Agrupar
Crtl' + 'Backspace' → Desagrupar
● 'Crtl' + click ou '2 clicks'→ Seleção de 1 elem do grupo
* Posso ter grupo dentro de grupo → arrastar entre camadas ou Crtl' + G

'Crtl' + click dir → mostra o parentesco do elem nas camadas (acima e abaixo dele)

'Shift' + click (cima ou baixo) → Move em linha reta (no mesmo alinhamento atual)
'Shift' + setas → Move 10px

Aula - Imagens

● imagem → = uma forma no Figma

Imagem → opções

Fill → a imagem espande e tente exiber apenas aquilo que o tam da forma delimita de espaço
Fit → mantém toda a imagem exibida, independete do espaço delimitado da forma
Crop → ajusta a imagem p exibir uma região específica dela
Tile → cria repetições da imagem
